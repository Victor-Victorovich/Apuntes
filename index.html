Proyecto blah blah blah.¶

El proyecto en el que nos hemos embarcado entraña un complejidad importate debido a la necesidad de compaginar el funcionamiento de diferentes equipos y subsistemas para conseguir resultados adecuados. Para ellos se intenta estructurar en este manual el contenido necesario para un seguimiento rápido de los distintos pasos a seguir así como para una mejor organización de la amplia documentación que se debe consultar.

This is a small sample book to give you a feel for how book content is structured. It shows off a few of the major file types, as well as some sample content. It does not go in-depth into any particular topic - check out the Jupyter Book documentation for more information.
